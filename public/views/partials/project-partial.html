This is Chaser, a project I was a part of while I was at the Rutgers Bootcamp. The goal is to help users find limited release and seasonal beers and keep track of their release dates. We used the breweryDB API to help build this basic proof of concept version. The project is was built using paid API access which has since lapsed, but feel free to look over our code on Github
Technologies used:
  • JavaScript
  • MongoDB
  • Angular.js
  • Express.js
  • Node.js
